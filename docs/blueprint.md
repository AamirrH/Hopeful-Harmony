# **App Name**: Hopeful Harmony

## Core Features:

- Mood Input: Accept user input for their current mood, either through pre-defined buttons or a text input field.
- AI-Powered Advice Generation: Leverage a generative AI model to create personalized, hopecore responses based on the user's expressed mood. Uses the user's mood to prompt an LLM tool, guiding the LLM's selection and incorporation of relevant hopecore concepts to offer support and upliftment.
- Dynamic Response Display: Display the AI-generated advice/quotes in a clean, engaging chat interface.
- Clear Emotion Identification: Visually confirms the mood, via the buttons that express the moods.

## Style Guidelines:

- Primary color: Sunny gold (#F5C96F) to evoke warmth and optimism.
- Background color: Soft cream (#FDF6ED), a desaturated near-tone of the primary color, to create a soothing backdrop.
- Accent color: Minty aqua (#A3D2CA), an analogous color to the primary gold, but offering a contrasting cool tone, will be used for subtle interface elements and highlights.
- Headline font: 'Playfair', a modern serif with an elegant, high-end feel.
- Body font: 'PT Sans' sans-serif, to ensure readability and accessibility of advice and quotes.
- Use soft, welcoming icons representing different moods and emotions.
- Subtle, uplifting animations when advice is generated and displayed.
